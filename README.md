# 🌾 هيباغري — HibaAgri

**المنصة الفلاحية الجزائرية المتكاملة**

منصة رقمية تجمع كل الخدمات الفلاحية في مكان واحد:
- 🚜 تأجير المعدات الفلاحية
- 🚛 نقل البضائع والمحاصيل
- ♻️ سوق النفايات العضوية
- 🏗️ آلات الأشغال العمومية
- 🏭 مستودعات التبريد
- 🤖 تشخيص الأمراض بالذكاء الاصطناعي
- 📦 المستلزمات الفلاحية
